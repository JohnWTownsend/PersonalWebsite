Programming is one of my favorite hobbies and now even a career for me, so check out some of my projects!


Technical Skills

JavaScript
JavaScript is perhaps my most go-to language for my projects, the way it allows you to manipulate the DOM is so powerful and fun.
C#
C# is mostly my favorite language. It is robust and easy to work with for the most part. I have used it as a backend in various ASP.Net applications as well as for WPF and Console Apps.
C
C was my first language, and thus has a special place in my heart. I enjoy using C for it's efficiency and I think it is great for systems type development but that isn't exactly my forte! As C is a rather low level language compared to it's predecessors, it doesn't see as much use from me, but I can still whip some C code up if needed.
Python
Can't leave out the good old snake! Python is such a great scripting language. I love using it for pounding out small programming tasks. I tend to stray away from it for projects of larger size. The larger a Python program gets, the harder it gets to read, modify, and add onto.
SQL
I have experience with SQL mainly in the context of direct data management and C# applications. I love working with data! It adds another dimension to projects through added persistence. Woo data!!!
Raspberry Pi
